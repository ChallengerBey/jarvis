# J.A.R.V.I.S Assistant

Bu proje, bilgisayarınızı sesli komutlarla kontrol etmenizi sağlayan bir PC asistanıdır. Groq AI ve yerel bir Python sunucusu (HACI) kullanır.

## Özellikler

- Sesli komutlarla program açma/kapama.
- Ses ve medya kontrolü.
- Groq AI ile doğal dil işleme.
- Ekran paylaşımı ve analitik arayüz.

## Kurulum

1. Bağımlılıkları yükleyin:
   `npm install`
2. Python sunucusunu (`flask_server.py`) çalıştırın.
3. Uygulamayı başlatın:
   `npm run dev`
4. Ayarlar panelinden Groq API Key ve İstasyon URL (Python sunucusu) bilgilerini girin.
